# Frontend Basics Practice

HTML, CSS, and JavaScript fundamentals practiced during the Software 
Development program at Code Academy (Baku, Sept 2022 – Jan 2023).

## Contents

- **html-css-basic** — Layout and styling exercises with HTML/CSS.
- **web-design-demo** — Additional HTML/CSS layout practice.
- **js-movie-site** — A movie listing site built with vanilla JavaScript.
- **flag-guessing-game-js** — A simple flag-guessing game using JavaScript.
- **custom-music-platform-demo** — A Spotify-style music platform layout 
  exercise.

> Note: these are archived coursework projects, not maintained or deployed.
